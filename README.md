# 30-Day Offensive Web Security Curriculum

An intensive, performance-optimized sprint dedicated to mastering application-layer vulnerabilities, manual exploitation techniques, and production-grade engineering remediations.

## ⚡ The Execution Formula
Every entry in this repository represents a 3-hour deep work block divided into three distinct operational phases:
1. **Theory & Mechanics (45 Mins):** Deconstructing target protocols and request states via PortSwigger Academy.
2. **Active Exploitation (90 Mins):** Intercepting, manipulating, and executing manual payloads using Burp Suite and WSL2.
3. **Defensive Documentation (45 Mins):** Drafting structural walkthroughs and writing secure remediation controls.
